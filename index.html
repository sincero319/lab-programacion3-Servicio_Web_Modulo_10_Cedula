Validador de cédula (módulo 10)

Servicio educativo: comprueba el dígito verificador según el algoritmo usado en la Cédula de Identidad y Electoral. Interfaz clara con salida en JSON y XML.

Ingreso

Formato sugerido: 000-0000000-0 (11 dígitos en total).

Importante: esto valida solo el algoritmo matemático (módulo 10), no si la persona existe en el padrón de la JCE. La JCE no publica una API pública gratuita para esa consulta; los proveedores comerciales (p. ej. Verifik) ofrecen KYC de pago.
API local (JSON / XML) Con el servidor Node en ejecución: GET http://localhost:3000/api/cedula?numero=40212345678 GET http://localhost:3000/api/cedula.xml?numero=40212345678 POST http://localhost:3000/api/cedula · body JSON {"numero":"40212345678"}
